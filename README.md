# 🌱 Paradise Nursery App

A simple and interactive e-commerce-style web application built using **React** and **Redux Toolkit**, where users can browse plants and add them to a cart.

---

## 🚀 Features

- 🌿 Browse a list of nursery plants
- 🛒 Add plants to cart
- ➕ Increase / decrease quantity
- ❌ Remove items from cart
- 💰 Dynamic total price calculation
- 📄 About Us section

---

## 🛠️ Technologies Used

- React (Functional Components + Hooks)
- Redux Toolkit (State Management)
- JavaScript (ES6)
- Tailwind CSS (basic styling)
- CSS
